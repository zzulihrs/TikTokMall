# TikTokMall



backend: 后端

脚本启动需要安装 `air`
启动脚本：
```shell
cd backend
make start
```


frontend: 前端

node 版本 v14.17.5

```shell
cd frontend
npm install
npm run serve
```



## 一、项目概述
本项目致力于打造一个功能全面、开发与部署流程规范的 AI 智慧电商购物平台，涵盖用户认证、用户管理、商品管理、购物车、订单处理、支付结算（支持银行卡、支付宝）以及LLM智能客服等功能模块，旨在为用户提供流畅的购物体验和智能化的服务。
